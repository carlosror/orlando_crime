General

The data is for Orlando Police Department dispatch calls rather than actual crimes. The data was obtained from the Code for Orlando data sources website

To safeguard citizens' privacy, each of the individual locations was randomly dithered so that none of the results shown is an exact location but rather an approximate one.

Either type a location's address or click on the map to obtain results around that location.

Clicking on an individual marker will show the crime type and date and time at .

Bar plots

The bar plots (and the density maps and the summary) are faceted according to the input selection. In order to keep the tables readable, the top six crime types are used.

Density maps

The density maps take a few seconds to display. They are faceted similarly to the way the bar plots are.

Table

The Table tab does what one expects it to: it tabulates the crime types, faceted in the same way the bar plots are.
